# 更新日志

## 2026-02-12 v2.0

### 新增功能
- ✅ 签名缓存（60秒内复用）
- ✅ 备用API端点（3个端点自动切换）
- ✅ 连接池管理
- ✅ 错误分类处理
- ✅ 统计信息（请求/成功/失败/重试）
- ✅ 指数退避重试

### 改进
- 网络重试：3次 → 5次
- 超时时间：20秒 → 30秒
- 错误处理：添加详细错误码

### 使用方法
```bash
# 查看统计
from okx_api import client
stats = client.get_stats()
print(f"请求: {stats['requests']}, 成功: {stats['success']}, 失败: {stats['failed']}")
```
